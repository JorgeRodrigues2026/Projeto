## 🚀 Como utilizar

### 1\. Clone o repositório

```bash
git clone <URL\_DO\_REPOSITORIO>
cd <NOME\_DO\_REPOSITORIO>
```

### 2\. Crie um ambiente virtual (opcional, mas recomendado)

**Podem fazer essa etapa diretamente pelo VS Code, como vimos em aula, ou seguir desta forma:**

#### Linux / macOS

```bash
python -m venv venv
source venv/bin/activate
```

#### Windows

```bash
python -m venv venv
venv\\Scripts\\activate
```

### 3\. Instale as dependências

```bash
pip install -r requirements.txt
```

### 4\. Execute o VS CS Code

Depois disso, abra o arquivo:

```bash
Mini Projeto\_salesinsight.ipynb
```
